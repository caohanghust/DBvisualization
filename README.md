#数据库数据可视化系统
###系统简介
    本系统主要用于解决数据库数据统计以及可视化问题，支持远程连接数据库展示数据，数据统计制图，原始数据制图，数据筛选，结合地图制图等。
###系统架构
    后台：PHP
    前端：angular + bootstrap + echarts
